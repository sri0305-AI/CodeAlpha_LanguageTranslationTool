
# Language Translation Tool

## Overview
This project is a Language Translation Tool that translates text from one language to another using Google Translate API.

## Features
- Translate text between multiple languages
- Simple and user-friendly interface
- Fast and accurate translations
- Built with Python and Gradio

## Technologies Used
- Python
- Gradio
- Googletrans

## Installation

pip install -r requirements.txt

## Run the Application

python app.py
